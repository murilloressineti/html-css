Principais Formatações

Negrito / Destaque

Nessa frase, temos um em negrito usando a tag <b> (não semântica).

Nessa frase, temos um termo em destaque usando a tag <strong> (semântica).

Itálico / Ênfase

Nessa frase, temos um termo em itálico usando a tag <i> (não semântica).

Nessa frase, temos uma termo em ênfase usando a tag <em> (semântica).

Texto marcado

Podemos criar um texto marcado utilizando a tag <mark>

Outra opção de texto marcado utilizando style.

Texto grande e pequeno

Estamos criando um texto grande utilizando a tag <big> (está obsoleto), e um texto pequeno nesse parágrafo utilizando a tag <small>.

Texto deletado

Podemos marcar um texto como excluído utilizando a tag <del>, para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido utilizando a tag <ins>, para dar uma ênfase e indicar que ele foi adicionado depois.

Existe também o sublinhado, utilizando a tag <u> (não semântica).

Texto sobrescrido

Para inserir x20 + 3, utilizando a tag <sup>

Texto subscrito

Para inserir H2O, utilizando a tag <sub>
